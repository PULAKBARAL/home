  1. Push Perfect Squares to a Stack

    You have stored a list containing ten integers. Thereafter, create a program with separate user-defined functions to perform the following operations based on this list:

    • Traverse the content of the list and push all the perfect square numbers into a stack.
    • Pop and display the content of the stack.
    For example: If MyList = [16, 44, 55, 64, 80, 81, 49, 66, 99, 75] 

    import math

def is_perfect_square(n):
    """Checks if a number is a perfect square."""
    if n < 0:
        return False
    sqrt_n = int(math.sqrt(n))
    return sqrt_n * sqrt_n == n

def push_perfect_squares(stack, number_list):
    """Pushes perfect squares from a list onto a stack."""
    print("Pushing perfect squares...")
    for num in number_list:
        if is_perfect_square(num):
            stack.append(num)
            print(f"Pushed {num}")
    return stack

def display_stack(stack):
    """Pops and displays the content of the stack."""
    print("\nPopping and displaying stack content:")
    while stack:
        print(stack.pop())

# Main Program
MyList = [16, 44, 55, 64, 80, 81, 49, 66, 99, 75]
square_stack = []

push_perfect_squares(square_stack, MyList)
display_stack(square_stack)

  2. Find Largest and Smallest in a Stack

    Write a user-defined function Large_Small() to create a stack for storing some numbers till the user wants. Display the content of the stack along with the largest and the smallest numbers in the stack.

    def Large_Small():
    """Creates a stack, finds, and displays the largest and smallest numbers."""
    stack = []
    while True:
        choice = input("Enter a number to push or 'done' to finish: ")
        if choice.lower() == 'done':
            break
        try:
            stack.append(int(choice))
        except ValueError:
            print("Invalid input. Please enter a number or 'done'.")
    
    if not stack:
        print("\nStack is empty.")
        return

    print("\nContent of the stack:", stack)
    
    largest = max(stack)
    smallest = min(stack)
    
    print("Largest number in the stack:", largest)
    print("Smallest number in the stack:", smallest)

# Run the function
Large_Small()

  3. Display Vowels from a String using Stack

    Write a user-defined function def Vowels() to input a string using a stack. Keep popping out each element from the stack and if it is a vowel then display, otherwise ignore.
    Sample Input: Information -> Sample Output: o, i, a, o, I

    def Vowels():
    """Inputs a string, pushes to a stack, and pops to display only vowels."""
    input_string = input("Enter a string: ")
    stack = list(input_string) # Push all characters onto the stack
    vowels = "aeiouAEIOU"
    
    print("\nVowels from the string (in reverse order):")
    output = []
    while stack:
        char = stack.pop()
        if char in vowels:
            output.append(char)
    
    print(", ".join(output))

# Run the function
Vowels()

  4. Reverse a String and Check for Palindrome

    Write a user-defined function def Reverse() to accept a string. Now, create a stack using the characters (beginning to the end) from the given string. Pop out the characters from the stack to form the reverse string. Finally, check and display whether the given string is a Palindrome or not.

    def Reverse():
    """Reverses a string using a stack and checks if it's a palindrome."""
    original_string = input("Enter a string: ")
    stack = list(original_string)
    
    reversed_string = ""
    while stack:
        reversed_string += stack.pop()
        
    print("\nOriginal String:", original_string)
    print("Reversed String:", reversed_string)
    
    if original_string.lower() == reversed_string.lower():
        print("The string is a Palindrome.")
    else:
        print("The string is not a Palindrome.")

# Run the function
Reverse()

  5. Display Names with Same First and Last Character

    Write a user-defined function def First_Last() to push any five names in a stack. Pop out each name from the stack but display only those names whose first and last characters are the same.

    def First_Last():
    """Pushes 5 names to a stack, then pops and displays names
    where the first and last characters are the same."""
    stack = []
    print("Enter 5 names:")
    for i in range(5):
        name = input(f"Enter name {i+1}: ")
        stack.append(name)
        
    print("\nNames with the same first and last character:")
    while stack:
        name = stack.pop()
        if len(name) > 0 and name[0].lower() == name[-1].lower():
            print(name)

# Run the function
First_Last()

  6. Push Car Names Based on Price

    Pranay has created a dictionary containing five different car names and their prices as key-value pairs. Write a program, with separate user-defined functions to perform the following operations:

    • Push the keys (car names) of the dictionary into a stack, where the corresponding value (price) is greater than or equal to 900000.
    • Pop and display the content of the stack.

    def push_expensive_cars(stack, car_dict):
    """Pushes car names with price >= 900000 to the stack."""
    print("Checking cars and pushing to stack if price >= 900000...")
    for car, price in car_dict.items():
        if price >= 900000:
            stack.append(car)
            print(f"Pushed '{car}'")
    return stack

def display_car_stack(stack):
    """Pops and displays the content of the car stack."""
    print("\nPopping and displaying expensive cars from the stack:")
    if not stack:
        print("No cars met the price criteria.")
        return
    while stack:
        print(stack.pop())

# Main Program
car_prices = {
    "Swift": 750000,
    "Fortuner": 3500000,
    "Creta": 1200000,
    "i20": 850000,
    "Thar": 1500000
}

push_expensive_cars(expensive_car_stack, car_prices)

  7. Stack Operations for Temperatures

    Write a Python code to implement a stack to store temperatures in degree Celsius. Perform the following operations using user's choice:

    • Push(MyStack, tmp): To push the temperature (tmp) into the stack.
    • Peek(MyStack): Displays the topmost temperature of the stack.
    • Display(MyStack): Displays the stack elements.

    def Push(MyStack, tmp):
    """Pushes a temperature onto the stack."""
    MyStack.append(tmp)
    print(f"Pushed {tmp}°C to the stack.")

def Peek(MyStack):
    """Displays the topmost temperature without removing it."""
    if not MyStack:
        print("Stack is empty. Cannot peek.")
    else:
        print("Topmost temperature:", MyStack[-1], "°C")

def Display(MyStack):
    """Displays all temperatures in the stack."""
    if not MyStack:
        print("Stack is empty.")
    else:
        print("Temperatures in stack (top to bottom):")
        for temp in reversed(MyStack):
            print(temp, "°C")

# Main Program Loop
temp_stack = []
while True:
    print("\n--- Temperature Stack Menu ---")
    print("1. Push a temperature")
    print("2. Peek at the top temperature")
    print("3. Display all temperatures")
    print("4. Exit")
    
    choice = input("Enter your choice (1-4): ")
    
    if choice == '1':
        try:
            temp = float(input("Enter temperature in Celsius: "))
            Push(temp_stack, temp)
        except ValueError:
            print("Invalid input. Please enter a number.")
    elif choice == '2':
        Peek(temp_stack)
    elif choice == '3':
        Display(temp_stack)
    elif choice == '4':
        print("Exiting program.")
        break
    else:
        print("Invalid choice. Please select from 1-4.")

  8. Stack Operations for City Names

    Implement a stack to manage different city names with the following functions based on user's choice:

    • MakePush(MyStack): To push a city name at the top of the stack and display the index of the top, the top city name, and all city names in the stack.
    • MakePop(MyStack): To remove a city name from the top of the stack. If the stack is empty, display "Underflow", otherwise display the popped item.
    Under what principle does the above entity work? 

    def MakePush(MyStack):
    """Pushes a city name and displays stack status."""
    city = input("Enter city name to push: ")
    MyStack.append(city)
    top_index = len(MyStack) - 1
    top_city = MyStack[-1]
    print(f"Top index: {top_index}")
    print(f"Top city: {top_city}")
    print("Current Stack:", MyStack)

def MakePop(MyStack):
    """Pops a city name and displays it, or shows underflow."""
    if not MyStack:
        print("Underflow: Stack is empty.")
    else:
        popped_city = MyStack.pop()
        print("Popped item:", popped_city)

# Main Program Loop
city_stack = []
while True:
    print("\n--- City Stack Menu ---")
    print("1. Push a city")
    print("2. Pop a city")
    print("3. Exit")

    choice = input("Enter your choice (1-3): ")

    if choice == '1':
        MakePush(city_stack)
    elif choice == '2':
        MakePop(city_stack)
    elif choice == '3':
        print("Exiting program.")
        break
    else:
        print("Invalid choice. Please select from 1-3.")

print("\nThe stack works on the LIFO (Last-In, First-Out) principle.")
