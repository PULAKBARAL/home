Python Basics Part II: Control Flow & Data Structures

Now that we have the basic building blocks—variables, data types, and operators—we can start building more interesting programs. This part will teach you how to control the flow of your program's execution with conditions and loops, and how to manage collections of data using Python's powerful built-in structures.

1. Input and Output Statements

A program isn't very useful if it can't interact with the user. Output is how the program presents information to the user, and input is how the program receives information from the user.

The print() Function in Detail

We've already used the print() function to display simple messages. It's a versatile function with some useful options called parameters.

  • Printing Multiple Items: You can print multiple values by separating them with commas. By default, print() will insert a space between them. 
    name = "Sonia"
age = 16
print("Name:", name, "Age:", age)
# Output: Name: Sonia Age: 16
  • The sep Parameter: The sep parameter allows you to specify a different separator instead of the default space. 
    print("08", "08", "2024", sep="-")
# Output: 08-08-2024

print("one", "two", "three", sep="***")
# Output: one***two***three
  • The end Parameter: By default, after a print() statement finishes, it moves the cursor to a new line. The end parameter lets you change this behavior. 
    print("Hello", end=" ") 
print("World!")
# Output: Hello World!

print("The first part", end=", and ") 
print("the second part.")
# Output: The first part, and the second part.

The input() Function

To get data from the user, we use the input() function. You can provide a string prompt inside the parentheses, which will be displayed to the user.

name = input("Please enter your name: ") 
print("Welcome,", name)

When this code runs, it will display the prompt, wait for the user to type something and press Enter, and then store the user's entry in the name variable.

Type Conversion (Casting): The Most Important input() Rule

The most critical rule to remember about input(): it always returns the data as a string (str), no matter what the user types. This can lead to one of the most common beginner errors.

Problem (leads to concatenation):

num1 = input("Enter the first number: ") # User enters 10
num2 = input("Enter the second number: ") # User enters 20
sum_result = num1 + num2 
print("The sum is:", sum_result)
# Expected: 30, Actual Output: The sum is: 1020

Analysis: The program didn't perform mathematical addition. Because input() returns strings, num1 was the string "10" and num2 was the string "20". The + operator, when used with strings, performs concatenation (joining them together).

Solution (with type casting):

num1_str = input("Enter the first number: ")
num2_str = input("Enter the second number: ")

# Convert the string inputs to integers
num1 = int(num1_str)
num2 = int(num2_str)
sum_result = num1 + num2
print("The sum is:", sum_result)
# Output: The sum is: 30

You can also perform the conversion in one line, which is a common practice:

num1 = int(input("Enter the first number: "))

2. Conditional Statements (if, if-else, if-elif)

So far, our programs have executed in a straight line, from top to bottom. This is called sequential flow. But to write truly useful programs, we need to be able to make decisions and execute different code based on certain conditions. This is called conditional flow. A key aspect of their syntax is indentation: the block of code to be executed under a condition must be indented.

The if Statement

The if statement is the simplest form of decision-making. It executes a block of code only if a given condition is True.

Syntax:

if condition:
    # This block of code runs only if the condition is True
    statement_1
    statement_2

Example:

num = int(input("Enter a number: "))
if num > 0:
    print("The number is positive.")
print("This line will always execute.")

The if-else Statement

The if-else statement provides an alternative path. If the condition is True, the if block runs. If it's False, the else block runs.

Syntax:

if condition:
    # Runs if condition is True
    block_A
else:
    # Runs if condition is False
    block_B

Example:

num = int(input("Enter an integer: "))
if num % 2 == 0:
    print(num, "is an even number.")
else:
    print(num, "is an odd number.")

The if-elif-else Chain

When you have more than two possibilities, you can use the if-elif-else chain. elif is short for "else if". Python checks the conditions from top to bottom and executes the first block where the condition is True. If no conditions are true, the final else block is executed (if it exists).

Clarifying elif vs. multiple ifs:

Using elif is crucial for checking mutually exclusive conditions. Consider assigning a grade based on marks.

Incorrect way (using multiple ifs):

marks = 95
if marks >= 90:
    print("Grade A") # This will print
if marks >= 80:
    print("Grade B") # This will also print!
if marks >= 70:
    print("Grade C") # This will also print!

This is logically incorrect because a student with 95 marks gets all three grades.

Correct way (using if-elif-else):

marks = 95
if marks >= 90:
    print("Grade A") # This prints, and the chain stops.
elif marks >= 80:
    print("Grade B")
elif marks >= 70:
    print("Grade C")
else:
    print("Grade D")

3. Loops (for, while)

Loops are used to execute a block of code repeatedly. This process is called iteration. Python provides two main types of loops: the for loop and the while loop.

Which Loop Should I Use?

  • Use a for loop when you know how many times you want to iterate or when you want to iterate over every item in a sequence.
  • Use a while loop when you want to loop as long as a certain condition is true, especially when you don't know the exact number of iterations.

The while Loop

A while loop repeats a block of code as long as its condition remains True.

count = 1
while count <= 5:
    print(count)
    count = count + 1 # Important: Update the loop variable!
# Output: 1 2 3 4 5

The for Loop and range()

A for loop is used for iterating over a sequence. The range() function is often used to generate a sequence of numbers for the loop.

  • range(stop): Generates numbers from 0 up to (but not including) stop.
  • range(start, stop): Generates numbers from start up to stop.
  • range(start, stop, step): Generates numbers from start to stop, incrementing by step.
# Iterate over a string
for character in "Python":
    print(character, end=' ')
# Output: P y t h o n 

# Using range()
for i in range(1, 6): # Generates numbers 1, 2, 3, 4, 5
    print(i)

Loop Control Statements

  • break: Immediately terminates the loop.
  • continue: Stops the current iteration and jumps to the beginning of the next one.
# Find the first number divisible by 7, but ignore multiples of 5.
for num in range(1, 21):
    if num % 5 == 0:
        continue  # Skip this iteration
    if num % 7 == 0:
        print("Found a number divisible by 7:", num)
        break     # Exit the loop
    print("Checking:", num)

Nested Loops

You can place one loop inside another. This is called a nested loop. It's commonly used for working with 2D data structures or for printing patterns.

# Outer loop controls the number of rows
for i in range(1, 6):
    # Inner loop controls the number of stars in each row
    for j in range(i):
        print("*", end="")
    print() # Move to the next line after each row

4. Lists and Tuples

Python offers powerful data structures for grouping and organizing data. The two most fundamental sequence types are lists and tuples. Both can store an ordered collection of items, but they have one critical difference that defines their use.

Lists (list)

A list is an ordered collection of items enclosed in square brackets []. Lists are mutable, which means you can change their content after they are created—you can add, remove, or change items.

Creating a List:

empty_list = []
student_marks = [88, 92, 75, 99, 85]
mixed_list = ["Ravi", 11, "A", 95.5]

Accessing Elements (Indexing and Slicing):

You access elements using their index, starting from 0. Slicing works just like with strings to get a sub-list.

marks = [88, 92, 75, 99, 85]
print(marks[0])       # Output: 88 (first item)
print(marks[-1])      # Output: 85 (last item)
print(marks[1:4])     # Output: [92, 75, 99] (items from index 1 to 3)

Modifying a List:

Because lists are mutable, you can change them directly.

marks = [88, 92, 75, 99, 85]
marks[2] = 80 # Change the value at index 2
print(marks)  # Output: [88, 92, 80, 99, 85]

Common List Methods:

  • append(item): Adds an item to the end of the list.
  • insert(index, item): Inserts an item at a specific position.
  • remove(item): Removes the first occurrence of an item.
  • pop(index): Removes and returns the item at a specific index (or the last item if no index is given).
  • sort(): Sorts the list in place.
  • reverse(): Reverses the order of the list in place.
  • index(item): Returns the index of the first occurrence of an item.
  • count(item): Returns how many times an item appears in the list.
scores = [90, 80, 95, 80]
scores.append(85)      # scores is now [90, 80, 95, 80, 85]
scores.sort()          # scores is now [80, 80, 85, 90, 95]
scores.remove(80)      # scores is now [80, 85, 90, 95] (only the first 80 is removed)

Tuples (tuple)

A tuple is an ordered collection of items enclosed in parentheses (). Tuples are immutable, meaning once a tuple is created, you cannot change its content.

Creating a Tuple:

empty_tuple = ()
point = (10, 20) # Represents an (x, y) coordinate
student_data = ("Ravi", 11, "A", 95.5)
# Parentheses are optional for creation
another_tuple = 1, 2, 3

To create a tuple with a single element, you must include a trailing comma: single_item_tuple = (50,). Without the comma, Python would think (50) is just the number 50.

Accessing Elements:

Indexing and slicing work exactly the same for tuples as they do for lists.

student_data = ("Ravi", 11, "A", 95.5)
print(student_data[0])   # Output: Ravi
print(student_data[1:3]) # Output: (11, 'A')

Immutability in Action:

If you try to change an element in a tuple, Python will raise a TypeError.

point = (10, 20)
# point[0] = 15  # This will cause a TypeError!

List vs. Tuple: A Detailed Comparison

The choice between a list and a tuple is a key design decision in Python. While they seem similar, their difference in mutability leads to different use cases, performance characteristics, and capabilities.

Feature List Tuple

When to Use a List vs. When to Use a Tuple:

  • Use a LIST when: You have a collection of similar items (e.g., a list of student names). You expect the collection to grow, shrink, or have its items change over time.
  • Use a TUPLE when: You want to store a collection of items that represents a single, fixed record (e.g., student = ("Aditya Sharma", 12345, "XI-B")). You want to ensure that the data cannot be accidentally changed. You need a slight performance boost or need to use the collection as a key in a dictionary.

5. Strings and String Methods

In Python, a string is an immutable sequence of characters. This means it behaves much like a tuple—it's ordered, you can access its characters using indexing and slicing, but you cannot change it in place.

String Traversal and Operations

You can process a string character-by-character using a for loop (traversal). You can also use + for concatenation and * for repetition.

my_word = "Python"
print(my_word[0])   # Output: P
# my_word[0] = 'J'  # This will raise a TypeError

Essential String Methods

Python's str class has a rich set of methods for manipulating strings. These methods return a new string and do not modify the original.

Understanding find() vs. index()

Use find() when you are not sure if the substring exists to avoid crashing your program. Use index() when your logic assumes the substring must exist; an error will alert you to logical flaws.

6. Dictionaries (dict)

A dictionary is an unordered collection of key-value pairs. They are enclosed in curly braces {}. Dictionaries are mutable and are highly optimized for retrieving values when you know the key.

Creating and Accessing Dictionaries

# A dictionary mapping fruit to their prices
fruit_prices = {"apple": 1.50, "banana": 0.75, "orange": 1.20}

# Accessing a value by its key
print(fruit_prices["banana"]) # Output: 0.75

# Using the get() method to avoid errors for non-existent keys
print(fruit_prices.get("mango")) # Output: None
print(fruit_prices.get("mango", "Not available")) # Output: Not available

Modifying a Dictionary

You can add, update, and remove items from a dictionary.

student_grades = {"Math": 90, "Science": 85}

# Add a new item
student_grades["English"] = 92
# Update an existing item
student_grades["Math"] = 95
# Remove an item using del
del student_grades["Science"]
# Remove an item using pop()
removed_value = student_grades.pop("Math")
print(student_grades) # Output: {'English': 92}

Iterating Through a Dictionary

The most common way to iterate is by using the .items() method to get both the key and the value.

student_scores = {"Alice": 95, "Bob": 88, "Charlie": 92}

for name, score in student_scores.items():
    print(f"{name}: {score}")

The final step in learning is to apply your knowledge. This section provides solved examples and a list of practice questions aligned with the CBSE practical syllabus to help you prepare for your exams and build your problem-solving skills.

Solved Example 1: String Analysis

# Program to analyze a user-inputted string.
# It counts the number of vowels, consonants, uppercase, and lowercase letters.
line = input("Enter a string: ")
vowel_count = consonant_count = uppercase_count = lowercase_count = 0
digit_count = space_count = 0
vowels = "aeiouAEIOU"

for char in line:
    if char.isalpha():
        if char.isupper():
            uppercase_count += 1
        else:
            lowercase_count += 1
        if char in vowels:
            vowel_count += 1
        else:
            consonant_count += 1
    elif char.isdigit():
        digit_count += 1
    elif char.isspace():
        space_count += 1

print("\n--- String Statistics ---")
print(f"Total Vowels: {vowel_count}")
print(f"Total Consonants: {consonant_count}")
print(f"Total Uppercase Characters: {uppercase_count}")
print(f"Total Lowercase Characters: {lowercase_count}")
print(f"Total Digits: {digit_count}")
print(f"Total Spaces: {space_count}")

Solved Example 2: Factorial of a Number

# Program to calculate the factorial of a non-negative integer.
num = int(input("Enter a non-negative number: "))
factorial = 1

if num < 0:
    print("Factorial does not exist for negative numbers.")
elif num == 0:
    print("The factorial of 0 is 1.")
else:
    for i in range(1, num + 1):
        factorial = factorial * i
    print("The factorial of", num, "is", factorial)
